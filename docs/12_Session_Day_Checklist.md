# Session Day Quick Reference Checklist

*San Diego County Amateur Radio Council | The Coolest VEC | vec@sandarc.org*

---

## Session Day Quick Reference Checklist

### Before Candidates Arrive

- [ ] Confirm all VEs present are SANDARC-accredited (check with Team Lead if unsure)
- [ ] Confirm session number has been obtained from SANDARC
- [ ] ExamTools session created and accessible; for online sessions, SANDARC President and VEC Manager added as Co-Owners
- [ ] Scratch paper, pencils, non-programmable calculators available (memories cleared)
- [ ] Testing area quiet and free from distractions; non-participants cleared from the area
- [ ] **NO recording equipment active** — video recording is prohibited at all SANDARC sessions
- [ ] No other VEC branding visible at the session location

### Candidate Check-In

- [ ] Greet each applicant warmly
- [ ] Verify photo ID — do not write down any numbers from the ID; do not photograph or video the ID
- [ ] For minors without photo ID: parent or guardian may verify identity in person
- [ ] Confirm FRN (not SSN) is on Form 605
- [ ] Confirm email address is on Form 605
- [ ] Confirm BQQ is answered; handle YES responses discreetly (see Section 6)
- [ ] Review any element credit documents — make a copy, return originals
- [ ] Add applicant to ExamTools roster
- [ ] Remind applicant: SANDARC charges no exam fee, but the FCC charges a separate $35 fee after submission

### During the Exam

- [ ] Announce session policies to all applicants before exams begin
- [ ] Issue exam via ExamTools (locked tablet, computer, or printed)
- [ ] At least 3 VEs observing all candidates at all times — stop all testing if team drops below 3
- [ ] VEs not reading, on phones, or having unnecessary conversations
- [ ] **No recording of any kind** — including screen capture
- [ ] Grade each element immediately after completion; inform applicant of score
- [ ] If passed: allow applicant to attempt next element if desired
- [ ] If upgrade earned: all 3 qualifying VEs certify Section 2 of Form 605
- [ ] Issue CSCE via ExamTools (or manually for paper sessions) — all 3 VEs sign
- [ ] Advise upgrading applicants about temporary identifier (/AG, /AE) and FCC $35 fee
- [ ] Advise new licensees they cannot transmit until a license appears in FCC ULS

### After the Last Candidate Leaves

- [ ] Finalize session in ExamTools; download VEC Archive file
- [ ] Email VEC Archive file and supporting documents to vec@sandarc.org
- [ ] Retain your copy of all session records until SANDARC confirms processing
- [ ] Record which exam version each applicant used (for retest tracking)
- [ ] Brief the team on what went well and what to improve for next time
